# Firebase 배워보자
## Firebase 설치
1. 파이서베이스를 설치하기 위해서는 https://nodejs.org 에서 node.js를 먼저 설치한 이후
2. 아래와 같이 터미널 창에서 -g(Global option)옵션으로 firebase를 설치한다.
```console
npm i -g firebase-tools
```
